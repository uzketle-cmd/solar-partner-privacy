3. Data Security & Retention â–¼

Security Measures

We implement industry-standard security practices to protect data:

  • Encryption: All data transmissions use HTTPS/TLS encryption
  • Access Controls: Strict authentication and authorization protocols
  • Regular Audits: Periodic security assessments and vulnerability scanning
  • Infrastructure Updates: Regular patching and system updates
  • Data Minimization: Only collect what's absolutely necessary

Data Retention Policy

  • Solar Coordinates: Processed in real-time, not stored
  • Project Data: Stored locally on your device, removable via app uninstall
  • Anonymous Analytics: Retained for 24 months, then automatically deleted
  • Crash Reports: Retained for 90 days for debugging, then purged

Deleting Your Data: Since we don't store personal information, there's no personal data to delete. Project data on your device can be removed by uninstalling the app or clearing app data through device settings.
